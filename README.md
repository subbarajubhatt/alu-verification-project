# 8-bit ALU Design & Verification

## Overview
This project implements and verifies an 8-bit Arithmetic Logic Unit (ALU) using Verilog and SystemVerilog.

## Features
- ADD (Addition)
- SUB (Subtraction)
- AND (Bitwise AND)
- OR  (Bitwise OR)
- XOR (Bitwise XOR)
- NOT (Bitwise NOT)
- Shift Left
- Shift Right

## Design (RTL)
The ALU is implemented using Verilog with a case statement based on opcode selection.

## Verification
A SystemVerilog testbench is used to:
- Apply different opcode test cases
- Simulate ALU operations
- Validate output results
- Generate waveform using VCD file

## Tools Used
- Verilog
- SystemVerilog
- EDA Playground
- EPWave (Waveform Viewer)
